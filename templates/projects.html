JSS Code

Technologies used: Spring boot, Thymeleaf, Hibernate, MySQL, HTML, CSS, jQuery, RabbitMQ, Docker, Amazon S3, Maven

December 2018 - February 2019

  • A platform developed to host programming contests.
  • The project has two main modules - judge-api which judges the submission and coding-platform.
  • Both the modules communicate using the REST APIs.
  • Users with Admin Role can create problems and contests with customized judging.
  • Registered users can participate in the contest, submit solutions to the problems and get real time judgement for their submissions.
  • Used Docker for creating multiple isolated environments to run the user’s submissions for various supported languages like Java, C++, Python, Go, etc.
  • Used RabbitMQ to queue the submissions. Executors fetch submissions from RabbitMQ to run them in the respective docker container.
  • The test-cases for the problems and source code, output and error streams of the submissions are stored on Amazon S3.
